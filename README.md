# Ball-game

A survival game where you collect orbs while evading an enemy, built in Python using Tkinter.

## Features
- Enemy AI that chases the player across the map
- Collectible orbs to earn currency
- Heal orbs and teleport orbs as power-ups
- Full upgrade shop (speed, health upgrades)
- Character colour customisation
- Text animation engine with skip functionality
- Username and password save/load system
- Dynamic health UI with colour changes
- Level progression with increasing difficulty

## Built with
- Python
- Tkinter
